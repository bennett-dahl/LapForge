Place app icons here:
- `icon.ico` (256x256 multi-size ICO for Windows)
- `icon.png` (512x512 PNG for taskbar/notifications)

Generate from a source SVG or PNG using a tool like https://icoconvert.com/
or electron-builder's icon auto-conversion (provide a 512x512 PNG as `icon.png`
and it will generate the ICO automatically).
